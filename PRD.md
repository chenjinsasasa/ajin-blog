# 阿锦的个人博客 - 产品需求文档 (PRD)

## 1. 项目概述

### 背景
阿锦（陈锦）需要一个个人博客来记录与AI助手谷子的日常对话进展、项目里程碑以及个人日记。目前这些内容散落在各处，缺乏统一的归档和展示平台。

### 目标
构建一个简洁、优雅的个人博客，支持MDX文章渲染、分类浏览、暗黑模式，并能在Vercel免费部署。

### 用户群体
- **主要用户**：阿锦（内容创作者）
- **次要用户**：访客（朋友、潜在合作者）

---

## 2. 用户故事

| 编号 | 用户故事 | 优先级 |
|------|----------|--------|
| US-001 | 作为阿锦，我希望按分类查看文章，以便快速找到特定类型的内容 | P0 |
| US-002 | 作为阿锦，我希望文章支持代码高亮，以便分享技术内容 | P0 |
| US-003 | 作为访客，我希望在手机上也能正常浏览，以便随时随地阅读 | P0 |
| US-004 | 作为阿锦，我希望支持暗黑模式，以便夜间舒适写作和阅读 | P1 |
| US-005 | 作为阿锦，我希望通过Markdown文件管理文章，以便使用熟悉的工具写作 | P0 |

## 3. 功能需求

### FR-001: 首页文章列表
- **描述**：展示最近发布的文章，支持按分类筛选
- **用户故事**：US-001
- **优先级**：P0
- **验收标准**：
  - 默认展示全部文章，按时间倒序排列
  - 支持切换筛选：「全部」「我们的进展」「阿锦的日记」
  - 每篇文章显示标题、摘要、发布日期、分类标签
  - 点击文章进入详情页

### FR-002: 文章详情页
- **描述**：渲染MDX内容，支持代码高亮和基础排版
- **用户故事**：US-002, US-005
- **优先级**：P0
- **验收标准**：
  - 正确渲染MDX内容（标题、段落、列表、链接、图片等）
  - 代码块支持语法高亮
  - 页面显示文章标题、发布日期、分类
  - 支持返回首页

### FR-003: 分类系统
- **描述**：文章归属两个固定分类
- **用户故事**：US-001
- **优先级**：P0
- **验收标准**：
  - 分类1：「我们的进展」- 记录阿锦与谷子的对话和项目进展
  - 分类2：「阿锦的日记」- 个人日记和随笔
  - 每篇文章必须属于且仅属于一个分类
  - 分类在文章Frontmatter中定义

### FR-004: 暗黑模式
- **描述**：支持系统跟随和手动切换两种模式
- **用户故事**：US-004
- **优先级**：P1
- **验收标准**：
  - 默认跟随系统偏好
  - 提供手动切换按钮（☀️/🌙）
  - 切换后主题状态持久化（localStorage）
  - 切换过程无闪烁

### FR-005: 响应式布局
- **描述**：适配桌面端和移动端
- **用户故事**：US-003
- **优先级**：P0
- **验收标准**：
  - 桌面端：内容区最大宽度768px，居中显示
  - 移动端：全宽显示，字体大小适配
  - 导航栏在移动端折叠为汉堡菜单
  - 图片自适应容器宽度

## 4. 数据结构

### 文章 Frontmatter 格式
```yaml
---
title: "文章标题"
date: "2026-04-03"
category: "我们的进展"  # 或 "阿锦的日记"
excerpt: "文章摘要，可选，默认取正文前150字"
---
```

### 文件目录结构
```
/content
  ├── progress/          # 我们的进展
  │   ├── hello-world.mdx
  │   └── project-update.mdx
  └── diary/             # 阿锦的日记
      ├── 2026-04-03.mdx
      └── weekend-thoughts.mdx
```

### 文章数据模型
```typescript
interface Post {
  slug: string;           // URL友好的标识
  title: string;          // 文章标题
  date: string;           // 发布日期 (YYYY-MM-DD)
  category: "progress" | "diary";
  categoryLabel: string;  // 显示用：「我们的进展」或「阿锦的日记」
  excerpt: string;        // 摘要
  content: string;        // MDX原始内容
}
```

## 9. 附录

### 9.1 分类说明
| 分类标识 | 显示名称 | 用途 |
|----------|----------|------|
| `progress` | 我们的进展 | 阿锦与谷子的对话记录、项目里程碑 |
| `diary` | 阿锦的日记 | 个人日记、随笔、生活记录 |

### 9.2 命名规范
- 文章文件名：`[日期]-[简短标题].mdx`，如 `2026-04-03-hello-world.mdx`
- Slug 生成：基于文件名，去除日期前缀，如 `hello-world`
