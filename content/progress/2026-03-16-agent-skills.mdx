今天做了一件积累性的事：给六个 Agent 配齐了各自的技能库。

来源是两个开源项目：
- **agency-agents**（48k ⭐）——角色定义参考
- **antigravity-awesome-skills**（24k ⭐）——具体技能实现

## 每个人各司其职

配完之后，每个 Agent 的技能是这样的：

- **阿龙**（前端）：Python 解释器、FastAPI 开发、Git 高级工作流、E2E 测试
- **蛋糕**（QA）：Playwright、差异审查、JavaScript/Python 测试模式
- **阿毛**（调研）：搜索专家、GitHub API
- **小U**（设计）：UI UX Pro Max、Shadcn UI、图片生成
- **小锦**（PM）：PRD 生成、Mermaid 图表、文档结构工具
- **阿商**（合规）：RAG 工程师、策略扫描

## 隔离原则

全局技能目录只有谷子能用，各 Agent 只能访问自己目录下的技能——不共用，不越权。

## 顺手做了健康检查

发现了 7 个配置问题，P0-P2 全部修复：阿商路径不存在、API Key 硬编码风险、蛋糕职责边界未定义……一条一条清完了。

技能配齐之后，感觉这支队伍才真正\"就位\"了。
