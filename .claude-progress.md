- 理由：个人博客单栏阅读场景，最强调内容优先、白空间、清晰层次感，WCAG AAA 级可访问性
- 变量：`--content-max-width: 680px`，`--spacing-large: 4rem`，`max-w-3xl` 居中单栏

**Color：Magazine/Blog — colors.csv Product Type #1**
- 来源：color 搜索 Result 1（Magazine/Blog）
- Light: `--bg: #FAFAFA` / `--fg: #09090B` / `--accent: #EC4899` / `--card: #FFFFFF` / `--border: #E4E4E7`
- Dark:  `--bg: #09090B` / `--card: #111113` / `--accent: #F472B6` / `--border: #27272A`
- 新增：`--accent-glow: rgba(236,72,153,0.08)`（卡片 hover 晕光）、`--accent-subtle`（tag 背景）

**Typography：Bold Typography Mobile（Inter-Tight Poster）— typography.csv #1**
- 来源：typography 搜索 Result 1
- Inter 800（H1 -0.04em tracking）/ Inter 700（H2 -0.03em）/ Inter 400-600（body）
- Playfair Display Italic（blockquote 引用段落）
- JetBrains Mono 400-500（code/pre）
- CSS Import: `Inter + JetBrains+Mono + Playfair+Display`（Google Fonts）
- Tailwind: `display: ['Inter'], quote: ['Playfair Display'], mono: ['JetBrains Mono']`

### 新增视觉细节
- `.accent-line`：渐变粉色装饰线（heading 前）
- `.prose h2::before`：4px 粉色竖条（sections 分区）
- `.prose hr::after`：`✦` 居中装饰符
- `.prose pre::before`：macOS 三色圆点（代码块质感）
- Card hover：`radial-gradient` accent-glow + `translateY(-2px)` + border-accent
- Header：`color-mix` + `backdrop-filter: blur(16px) saturate(160%)`
- `.animate-fade-up`：0.4s ease 页面入场动画
- Tag pills：`border: 1px solid` + 深色模式半透明背景
