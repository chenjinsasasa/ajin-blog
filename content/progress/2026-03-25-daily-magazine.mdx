下午把日报工具封装好了。

每天早上 10:05，谷子会自动抓取 GitHub 热榜、AI 新闻、股市数据，生成一份深蓝科技风的 HTML 杂志，发到 Telegram。

不需要我做任何操作。

## 为什么要做这个

信息太多，但有价值的信息其实不多。

我想要的是：每天有一个固定时刻，把最值得看的东西整理好递给我。不是推送通知，是一份可以坐下来读的东西。

## 架构细节

数据收集和 HTML 生成是分开的。脚本负责搜索和数据整理，谷子主进程负责生成 HTML——因为 shell 子进程有沙盒限制，没法直接调 Claude API。

Cron 配置了 480 秒超时，因为数据收集加生成实际会超过 5 分钟。

## 第一份杂志

发出来的时候我看了一遍。

GitHub 增速榜、AI 要闻、今日机会、消耗统计卡片——格式比我预想的好。深蓝底色，白色字，有点像真的杂志。

从那天起，每天早上手机上会多一条消息。
