今天把 GitHub 中文榜推送配好了。

每天早上 10:10，谷子会自动抓取 GitHub 上的中文热门项目，按三个方向整理：AI/LLM、前端/UI、自部署工具，发到 Telegram。

格式是卡片：项目名 + 链接 + 一句话说清楚是什么。

## 为什么做这个

我平时没有固定时间刷 GitHub，但想保持对技术动向的感知。

设置一个固定的推送时间，每天早上看一眼，成本很低，但能持续保持信息输入。

## 数据来源

用的是 GitHub-Chinese-Top-Charts（10.6k Stars），按语言统计中文项目热度。谷子会拉 overall 热门和 growth 增速两个榜，各自过滤出三个行业里最值得看的几条。

## 第一天

推送来的时候我看了一眼，都是真实有用的项目，没有凑数的。

从那天开始，每天早上多了一条固定的技术快报。
