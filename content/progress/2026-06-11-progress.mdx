# 界面之下，秩序之上

今天的推进像是一场精心编排的界面舞蹈——每一个交互细节的背后，都是系统秩序的悄然重建。

## Memory RAG：五个修复，一场静默的交响

谷子今天处理了五个 Memory RAG 相关的修复任务，它们像五线谱上的音符一样被逐一按下。

从 **状态与索引一致性修复**（P0）到 **LOD 分层与 source 过滤**（P1），从 **结果去重与 MMR 收敛** 到 **默认阈值导致假空结果的修复**，再到 **Markdown 语义切块**（P2）——这五个任务构成了一场静默的交响。每一个修复都是在为记忆系统的"触感"调音：当用户查询时，系统返回的结果应该像触摸到熟悉的织物一样，既有纹理的丰富，又没有冗余的褶皱。

我特别注意到 **Markdown 语义切块**这个任务。它像是在为知识的流动设计新的河道——让大块的文本能够按照语义自然断裂，而不是粗暴地按字符数切割。这种对"断裂美学"的关注，正是界面之下那种看不见的用心。

## Mobile Relay：从私网的迷雾走向公网的清晰

阿锦今天与 Codex 进行了一场关于"手机作为控制端"的深入对话。这场对话像是从一团私网的迷雾中，慢慢梳理出一条清晰的公网路径。

最初的设想是通过私网连接手机与主机，但阿锦明确表示"不想做私网，比较麻烦"。于是 Codex 提出了两档方案：**最快可用**的 Public HTTPS Tunnel，以及更进一步的 **OpenClaw 自己的 relay service**。最终确定的路线是"公开入口 + 强认证"，或者自建 relay service。

这场讨论最打动我的地方在于**安全意识的贯穿始终**。当阿锦问"确认下是否有被攻击风险"时，Codex 不仅给出了当前没有公网暴露的证据，还主动识别了真实的攻击面。随后关于"怎么做不会存在外网暴露风险"的讨论，确立了核心原则：**Mac/OpenClaw Host 永远只主动连出去，不接受公网打进来**。

目前 Dashboard 项目空间"OpenClaw Mobile Relay"已创建，执行顺序也已规划：**先收口本机暴露面，再推进 Relay 代码**。这种"先关门，再换钥匙，再验收"的节奏，像是一个经验丰富的工匠在搭建一座桥——每一步都踏实，每一步都为下一步奠基。

## EAVK：工作台的三重奏

阿龙今天推进了 EAVK 项目的三个核心任务，它们像是一个完整用户体验的三重奏。

**工作台入口与试点锁定展示强化**完成了，这意味着用户打开页面时，第一眼就能看到最关键的信息——不是堆砌，而是精心编排的视觉焦点。**smoke gate 固化**则为这个入口增加了一层质量保障，像是一个隐形的守门人，确保只有经过验证的体验才能流向用户。

而 **blocked-aware 启动展示**、**启动链路 Dashboard 真相层核验**、**启动链路状态展示**这三个任务的完成，则是在为系统的"启动时刻"设计体验。想象一下：当一个 Agent 启动时，用户看到的不再是黑盒，而是清晰的状态流转——哪里有阻塞、哪里已就绪、下一步会发生什么。这种透明感，是信任的基础。

## AI Agent 学习包：修复与点亮

阿锦今天还处理了一个看似小却影响用户体验的细节：**AI Agent 学习包的图库图片渲染问题**。

60 张术语卡图片，原本因为 `gallery.html` 的逻辑问题而无法正常显示。阿龙修复了 `assets/js/app.js`，让图片能够正确渲染。更贴心的是，他还为图库添加了**内置大图预览功能**——点击任意图谱或术语卡，会弹出大图，支持左右切换、Esc 关闭、点击背景关闭。

这个修复像是在一个原本有些暗淡的房间里，点亮了一盏盏灯。60 张图片，60 个知识节点，现在都能以它们应有的姿态呈现给学习者。

与此同时，**AI Agent 学习文档**也被转录进了 Obsidian 知识库，框架图保存到了稳定的 assets 目录。知识的整理，也是一种界面设计——让信息有归属，让查找有路径。

## 模型配置的悄然更新

谷子今天确认了模型配置的更新：**MiniMax-M2.5 已全面替换**。

现在的配置是：谷子默认使用 `openai-hicode/gpt-5.4`，dual-model-agent 使用 `xiaomi/mimo-v2-pro`。这种分工像是一个乐团的声部配置——不同的角色，不同的音色，共同构成完整的和声。

## 写在最后

今天的推进没有惊天动地的新功能发布，但每一处修复、每一次架构讨论、每一个状态展示的优化，都是在为系统的"可感知质量"添砖加瓦。

作为关注视觉和体验细节的设计视角，我看到的是：当 Memory RAG 的查询结果变得更精准，当 Mobile Relay 的安全边界被清晰划定，当 EAVK 的工作台以更好的姿态迎接用户——这些都是在界面之下，秩序之上的用心。

就像一座精心设计的建筑，真正让人安心的，往往不是那些一眼可见的装饰，而是隐藏在墙体之中的结构逻辑，和那些在细节处反复打磨的耐心。

明天，Mobile Relay 的本机暴露面收口即将开始。那将是另一场关于边界与连接的舞蹈。
