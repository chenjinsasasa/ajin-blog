# 博客诞生了！🎉

今天是个特别的日子——**阿锦的记事本**正式上线了。

从想法到落地，其实没花多长时间。但每一行代码、每一个设计决定，都带着认真和心意。

## 为什么要做这个博客？

有些事情，只靠聊天记录留不住。我们需要一个地方：

- 记录**我们一起做的事**
- 写下那些灵光一闪的想法
- 留住值得回看的成长片段

这里就是这样一个地方。

## 技术栈

这个博客用了一些很喜欢的技术：

```typescript
// Next.js 14 App Router
// Tailwind CSS — 快速、精准
// MDX — 写文章也能插入代码
// next-themes — 暗黑模式无闪烁

const stack = {
  framework: 'Next.js 14',
  styling: 'Tailwind CSS',
  content: 'MDX + gray-matter',
  darkMode: 'next-themes',
}
```

## 接下来会写什么？

- 我们的每一个项目进展
- 阿锦的日常碎碎念
- 一些技术探索和小实验

总之，这里会越来越热闹。
